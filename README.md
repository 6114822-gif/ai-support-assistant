# AI Support Assistant — ИИ-ассистент поддержки

Веб-чат на Python (Flask) с ассистентом поддержки на базе **Claude API**
(Anthropic). Демо для вымышленного интернет-магазина "TechStore" —
отвечает на вопросы про доставку, возврат, гарантию и оплату, опираясь
только на правила магазина, зашитые в промпт (не выдумывает от себя).

Раздел про возврат/обмен технически сложных товаров (телефоны, телевизоры,
компьютеры и т.п.) построен на реальных нормах Закона РФ "О защите прав
потребителей" (ст. 18, 25, 26.1) — формулировки сверены с официальным
текстом закона, а не придуманы. Ассистент различает: брак в первые 15 дней
(ст. 18, право на возврат/замену независимо от серьёзности поломки), брак
после 15 дней (только при существенном недостатке), и "просто передумал"
(другой перечень товаров, не подлежащих обмену). Каждый такой ответ
завершается дисклеймером и контактом поддержки — бот не заменяет юриста.

## Что показывает этот проект

- Как подключить Claude API в свой Python-проект (`anthropic` SDK)
- Как сделать простой веб-интерфейс чата (Flask + HTML/JS)
- Как ограничить ИИ-ассистента конкретной ролью и базой знаний через системный промпт
- Как заложить в промпт точную предметную область (в данном случае — нормы
  конкретного закона) и не дать модели путать похожие, но разные правила
- Как передать модели текущую дату, чтобы она сама считала интервалы
  (например, "сколько дней прошло с момента покупки"), а не переспрашивала

## Установка и запуск

```bash
cd ai-support-assistant
python3 -m venv venv
source venv/bin/activate   # на Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### Получи ключ Anthropic API

1. Зайди на [console.anthropic.com](https://console.anthropic.com)
2. Зарегистрируйся/войди, пополни баланс (используется по факту трат за токены — по чуть-чуть, не подписка)
3. В разделе **API Keys** нажми **Create Key**, скопируй ключ (вида `sk-ant-...`)

```bash
cp .env.example .env
```

Открой `.env` и вставь ключ:
```
ANTHROPIC_API_KEY=твой_ключ_сюда
```

### Запусти

```bash
python3 app.py
```

Открой в браузере: [http://127.0.0.1:5000](http://127.0.0.1:5000)

Попробуй спросить: "Сколько стоит доставка?", "Могу я вернуть товар?",
"Какая гарантия на технику?" — а также что-то не по теме (например,
"какая завтра погода?") — ассистент должен вежливо отказаться отвечать
и направить в поддержку.

## Как адаптировать под свой проект

- Замени текст в `SYSTEM_PROMPT_TEMPLATE` (файл `app.py`) на правила своего бизнеса.
- Название компании, стиль общения, язык ответов — всё настраивается через этот промпт.
- Для реального проекта стоит добавить: лимит на количество сообщений в минуту
  (защита от больших счетов за API), логирование вопросов, историю в БД вместо памяти браузера.
